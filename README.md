# multi-level-slider-game

A game where the player avoids obstacles while sliding along a strip and advancing to the next level upon completion.

Developed using Brackeys "How to make a Video Game in Unity" video as inspiration for a project in UBC's Geering Up Game Development workshop.

# How to Run

Make sure to have GitHub Desktop and Unity installed.

- Copy the repository URL from the dropdown under "<> Code"
- Open GitHub Desktop and clone a repository.
- Go to the URL tab, paste the repository URL, and hit "clone".
- Go to Unity Hub, click the dropdown for "Open", and press "Add project from disk."
- Select the folder labeled "multi-level-slider-game".
- After it has finished importing, open the game by clicking on the project labeled the same as the repository name.
- Press the play button at the top to start playing the game.

# Gameplay

- After pressing the play button, the player (blue cube) wil start sliding.
- To control it, use the left and right arrow keys.
- Dodge the obstacles while trying not to fall off the map.
- Upon completion, the game will move onto the next level.
- To stop, press the play button once again.
